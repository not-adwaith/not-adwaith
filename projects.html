Saltus

ALternative Hadoop

Created a system(Saltus) to tackle the volume of big data that is getting generated, right at the source which helps to avoid over-accumulation of data. Working implementation of DRIFE architecture for analyzing the content in the data. Mainly focused on implementing the 'v-pat' architecture that aims data reduction for its video identification operations. Built usinjg Artificial intelligence and Convolution Neural networks.

Big Data processing and storage is one of the major issues to be addressed in this current era of increasing use of digital infrastructure. In this paper, a novel method to tackle the issue of Big Data processing and a better technique to ’store what is only required’ is proposed. The key idea of this paper is to propose and discuss a possible architecture to store and process Big data using the concepts of Artificial Intelligence and Neural Networks. The paper introduces the processing technique called DRIFE and storage files called CFS. The main ideology of Saltus is to tackle the volume of big Data that is getting generated, right at the source which helps to avoid over accumulation of data over time.

Subtraction of Video Frames

The project was to perform background subtraction on videos using three methods namely frame differencing, Gaussian model and K-Nearest neighbors. When MSE and PSNR values were regarded as evaluation metrics, it showed that the KNN algorithm slightly outperformed the other methods. Video subtraction on video frames is a technique used to detect changes between two or more video frames. The technique involves subtracting the pixel values of one frame from another frame to obtain a difference image.

The difference image contains the pixels that have changed between the two frames, which can be used to identify moving objects, changes in lighting, or other changes that have occurred between the frames.

To perform video subtraction, the following steps can be taken:

Convert the video frames to grayscale to simplify the process.
Select two frames that are close in time.
Subtract the pixel values of one frame from the other frame.
Threshold the resulting difference image to obtain a binary image of the changes.
Apply morphological operations, such as erosion and dilation, to remove noise and smooth the binary image.
Repeat the process for subsequent frames to detect changes over time.
Video subtraction can be a useful technique in various applications, such as surveillance systems, traffic monitoring, and motion analysis. However, it can also be prone to errors and false positives, so it should be used in conjunction with other techniques and algorithms to improve its accuracy.

Multiplayer Quiz using Socket Programming

A multiplayer quiz game in a KBC show format is implemented in this project. The quiz can have any number of users. The game starts once maximum number of players have joined .TCP/IP was used to create the application stack. Python was chosen, due to it easy to code nature.

A socket program is a computer program that uses sockets, a set of programming interfaces for sending and receiving data over a network. Sockets provide a way for processes to communicate with each other on the same or different computers.
